Copy into Outlook (desktop)

1. Open this file in a browser → 2. Select everything inside the dashed orange box → 3. Copy (Ctrl/Cmd + C) → 4. Outlook → File ▸ Options ▸ Mail ▸ Signatures → create a signature → paste (Ctrl/Cmd + V) → Save.

Copy into Outlook on the web / Gmail / iPhone Mail

Select inside the dashed box, copy, then paste into the signature editor (Outlook web: Settings ▸ Mail ▸ Compose and reply; Gmail: Settings ▸ See all settings ▸ Signature; iPhone: easiest to set up the signature on desktop Outlook/Gmail, which syncs to mobile).

If Outlook blocks the logo

External images are sometimes blocked until the recipient clicks “Download pictures”. To minimize this: make sure https://inteka.pl/images/logo_final.jpg returns the image (open it directly in a browser), keep the image on the same domain as the website, and avoid GitHub raw / base64 links. Recipients can permanently allow images from your domain in their trust settings.
